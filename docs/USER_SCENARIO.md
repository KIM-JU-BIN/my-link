# 마이링크 (MyLink) 사용자 시나리오

## 1. 방문자: 마이링크 페이지 방문 과정

- 방문자는 특정 크리에이터의 모든 링크 모음을 한 곳에서 확인하기 위해 제공된 마이링크 고유 URL(`mylink.com/nickname`)로 접속한다.
- 방문자는 접속한 페이지의 주인이 누구인지 파악하기 위해 최상단에 노출된 프로필 이미지, 닉네임, 그리고 한 줄 소개를 읽는다.
- 방문자는 원하는 콘텐츠(블로그, SNS, 포트폴리오 등)로 이동하기 위해 파비콘 이미지가 포함된 직관적인 링크 버튼을 클릭한다.

---

## 2. 소유자: 링크 추가/수정/삭제 과정

- 소유자는 본인만의 고유한 프로필 페이지를 생성하고 접속하기 위해 구글 계정으로 소셜 로그인을 진행한다.
- 소유자는 새로운 공유 링크를 프로필에 등록하기 위해 대시보드 화면에서 '새 링크 추가' 버튼을 눌러 타겟 URL과 링크 제목을 입력한다.
- 소유자는 기존에 등록된 링크의 제목이나 URL 주소를 변경하기 위해 해당 링크 항목의 인라인 편집(Inline Edit) 기능을 활성화하여 즉시 텍스트를 수정하고 저장한다.
- 소유자는 더 이상 유효하지 않거나 노출하고 싶지 않은 링크를 제공하지 않기 위해 해당 링크 아이템의 '삭제' 버튼을 클릭하여 완전히 제거한다.

---

### 💡 개선점 제안 (추후 고도화 고려 사항)

현재 PRD스펙에 기반한 시나리오입니다만, 소유자(사용자)의 사용성을 더욱 높이기 위해 초기 기획에서 배제되었던 기능들에 대해 다음과 같은 개선점을 제안드립니다.

1. **링크 활성화/비활성화 (숨김) 토글 기능**
   - **이유**: 특정 기간에만 진행하는 이벤트 링크나 일시적으로 숨기고 싶은 링크를 매번 '삭제'했다가 나중에 다시 '추가'하는 것은 매우 번거롭습니다. 삭제 대신 시야에서만 가려둘 수 있는 '숨김/노출 토글' 스위치가 있다면 사용성이 크게 향상됩니다.
   - **시나리오**: "소유자는 특정 링크를 일시적으로 프로필에서 감추기 위해 해당 링크의 노출 토글 버튼을 '비활성' 상태로 변경한다."

2. **드래그 앤 드롭을 통한 순서 변경 기능**
   - **이유**: 가장 중요하게 홍보하고 싶은 링크(예: 최근 업로드한 유튜브 영상, 신규 서비스 오픈 등)를 맨 위로 올리고 싶을 때, 단순히 '생성일(createdAt)' 순서대로만 고정되어 있으면 원하는 대로 강조하기가 어렵습니다.
   - **시나리오**: "소유자는 가장 중요한 링크가 방문자 눈에 먼저 띄게 하기 위해 목록 상의 링크 블록을 마우스로 드래그하여 최상단으로 끌어올린다."
